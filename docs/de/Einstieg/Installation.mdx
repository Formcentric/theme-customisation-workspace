# Schnellstart

Folgen Sie diesen Schritten, um mit dem Formcentric Theme Customization Workspace loszulegen:

Stellen Sie sicher, dass [Node.js](https://nodejs.org/) und [pnpm](https://pnpm.io/) auf Ihrem System installiert sind:

```bash
# Node
node -v

# pnpm
pnpm -v
```

Überprüfen Sie die Datei `.nvmrc`, um die empfohlene Node.js-Version zu finden. Installieren Sie diese bei Bedarf und führen Sie anschließend folgende Befehle aus:

- `corepack enable`, um Corepack zu aktivieren
- `corepack prepare pnpm@latest --activate`, um pnpm einzurichten

Führen Sie danach die folgenden Befehle aus, um Ihre Entwicklungsumgebung einzurichten:

```bash
# Abhängigkeiten installieren
pnpm i

# Entwicklungsserver starten
pnpm dev-cloud # Für FC-Cloud-Nutzer
# oder
pnpm dev-local # Für self-hosted Nutzer
```

Sobald der Entwicklungsserver läuft, können Sie die Anwendung unter `http://localhost:3000` aufrufen.
Verwenden Sie den passenden Befehl, je nachdem, ob Sie FC Cloud oder einen self-hosted Formcentric Headless Server nutzen.

Weitere Konfigurationsmöglichkeiten und nächste Schritte finden Sie in den Guides zu [Nutzung](/docs/de-einstieg-nutzung--docs) und [Konfiguration](/docs/de-einstieg-konfiguration--docs).
