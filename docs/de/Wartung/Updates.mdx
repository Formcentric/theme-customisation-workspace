# Themes aktualisieren

Um die Basis-Themes, also die Standard-Templates, auf denen Ihr Theme basiert, in Ihrem Workspace zu aktualisieren, führen Sie den folgenden Befehl aus:

```bash
pnpm update-theme
```

Mit diesem Befehl aktualisieren Sie alle Basis-Themes im Verzeichnis `themes-base` auf die neuesten Versionen des Formcentric-Moduls.

### Checkliste nach der Aktualisierung

1. **Extended Formcentric-Themes**

    - Anpassungen von Styles über Variablen funktionieren in der Regel weiterhin.
    - Überprüfen Sie etwaige Überschreibungen von Templates auf neue erforderliche Attribute oder Funktionen.

2. **Full- und Custom-Themes**

    - Überprüfen Sie die Änderungen in den aktualisierten Basis-Themes.
    - Integrieren Sie notwendige Anpassungen manuell.

3. **Immer testen!**
    - Testen Sie Ihr Theme nach jeder Aktualisierung, um sicherzustellen, dass es mit den aktualisierten Komponenten wie erwartet funktioniert.
