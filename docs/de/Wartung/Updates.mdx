# Themes aktualisieren

Um die Base-Themes in deinem Workspace zu aktualisieren, führen Sie folgenden Befehl aus:

```bash
pnpm update-theme
```

Dieser Befehl aktualisiert alle Base-Themes im Verzeichnis `themes-base` mit den neuesten Versionen aus dem Formcentric-Modul.

### Post-Update Checkliste

1. **Extended FC Themes**

    - Style-Anpassungen über Variablen funktionieren in der Regel weiterhin.
    - Prüfen Sie etwaige Template-Overrides auf neue erforderliche Props oder Funktionen.

2. **Full/Custom Themes**

    - Überprüfen Sie die Änderungen in den aktualisierten Base-Themes.
    - Integrieren Sie notwendige Anpassungen manuell.

3. **Immer testen!**
    - Testen Sie Ihr Theme nach jeder Aktualisierung, um sicherzustellen, dass es mit den aktualisierten Komponenten wie erwartet funktioniert.
