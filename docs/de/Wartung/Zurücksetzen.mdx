# Workspace zurücksetzen

Mit dem Reset-Befehl können Sie Ihren Workspace aufräumen, indem Sie:

- Alle benutzerdefinierten Themes entfernen
- Das Verzeichnis für Basis-Themes leeren
- Wichtige `.gitkeep`-Dateien beibehalten
- Abhängigkeiten neu installieren

Um den Workspace zurückzusetzen, führen Sie folgenden Befehl aus:

```bash
pnpm reset
```

Sie werden vor dem Ausführen des Reset-Vorgangs zur Bestätigung aufgefordert.

## Was wird zurückgesetzt?

Welche Dateien und Verzeichnisse betroffen sind, können Sie in der Datei `formcentric.config.js` konfigurieren  
(Siehe [Reset-Konfiguration](/docs/de-einstieg-konfiguration--docs#reset-konfiguration)).

## Nach dem Zurücksetzen

Nach dem Ausführen des Reset-Befehls:

1. Ist der Workspace auf den ursprünglichen Zustand zurückgesetzt.
2. Wurden alle Abhängigkeiten automatisch neu installiert.
3. Können Sie direkt mit der Erstellung neuer Themes beginnen.
