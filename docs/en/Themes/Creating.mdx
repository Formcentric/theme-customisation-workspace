# Create your own theme

Create your own theme with the following command:

```bash
pnpm create-theme
```

### Extended Formcentric theme

- Ideal for targeted style adjustments to an existing theme
- Inherits most styles from the selected base theme
- Enables the targeted overriding of components and variables
- Lowest maintenance effort


### Full Formcentric theme

- A complete copy of an existing theme
- Full control over all theme components
- Higher maintenance effort if changes made to base theme
- Good balance between flexibility and complexity

## Best practices

1. **Modular structure**

    - Subdivide your styles into logical components
    - Use partials to improve maintainability
    - Implement a uniform naming convention

2. **Use of variables**

    - Use theme variables to ensure consistent styling
    - Make your styles configurable via `definition.json` (Formcentric Cloud) and the files `_fc-variables.scss`, `_variables.scss` and `_variables.json`
    - Use SCSS features to implement DRY principles

3. **Responsive Design**

    - Incorporate breakpoints for mobile devices and desktop
    - Use flexible units such as ‘em’
    - Run tests on various screen sizes

4. **Accessibility**
    - Take focus states into account
    - Make sure you have adequate colour contrasts
    - Provide keyboard navigation support for your user-defined templates
