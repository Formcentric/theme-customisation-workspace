# Updating themes

To update the base themes in your workspace – i.e. the standard templates your theme is based on – execute the following command:

```bash
pnpm update-theme
```

This command updates all of the base themes in the `themes-base` directory to the latest versions from the Formcentric module.

### Post-update checklist

1. **Extended Formcentric themes**
    - Style customisations with variables will normally work as before.
    - If you have overridden templates, check to see whether new attributes or functions are necessary.

2. **Full and custom themes**
    - Check the changes made to the updated base themes.
    - Integrate any necessary adjustments manually.

3. **Always test!**
    - After each update, test your theme to ensure that it works as expected with the updated components.
