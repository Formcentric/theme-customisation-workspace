# Updating Themes

To update base themes in your workspace, run:

```bash
pnpm update-theme
```

This command updates all base themes in your `themes-base` directory with the latest versions from the Formcentric module.

### Post-Update Checklist

1. Extended FC Themes:
    - Style customizations via variables typically continue working.
    - Check any template overrides for new required props or functionality.
2. Full/Custom Themes:
    - Review changes in updated base themes.
    - Manually incorporate necessary modifications.
3. Always test your themes to ensure they work as expected with the updated components.
